# MGS.MonoListener

## Summary

Listener for Unity MonoBehaviour.

## Ability

- Listener for Graphic and Pointer.
- Listener for Collider and Mouse.
- Listener for Collider and MRTK.
- Listener for Collider and XRI.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.MonoListener.git?path=/Assets
  ```
## Samples

- Unity --> Window --> Package Manager --> Packages-Mogoson --> Mono Listener --> Samples.
